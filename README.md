# Machine Downtime Analysis

## Overview

This project analyzes manufacturing machine downtime data to identify failure patterns, high-risk machines, and operational inefficiencies. The goal is to support better maintenance decisions and reduce production losses.

---

## Data Source

The dataset was sourced from Kaggle:

https://www.kaggle.com/datasets/srinivasanusuri/optimization-of-machine-downtime

The dataset contains information about machine operational conditions and downtime events. :contentReference[oaicite:0]{index=0}

---

## Data Description

The dataset includes:

- Machine operational status  
- Downtime duration  
- Maintenance events  
- Process parameters  

It captures machine performance and downtime records that can be used to analyze equipment reliability and production delays.

---

## Tools Used

- Excel — data cleaning and preprocessing  
- Power BI — dashboard creation and visualization  

---

## Analysis Performed

- Cleaned and structured raw downtime data  
- Analyzed downtime duration and frequency  
- Identified machines with highest downtime  
- Compared maintenance-related vs failure-related downtime  
- Explored patterns in machine performance  
- Built an interactive dashboard in Power BI  

---

## Key Insights

- A small number of machines contribute disproportionately to total downtime  
- Repeated failure patterns suggest preventable issues  
- Maintenance-related downtime can be distinguished from failure-driven downtime  
- Downtime trends indicate opportunities for predictive maintenance  

---

## Business Impact

This analysis can help:

- Reduce unplanned downtime  
- Improve maintenance scheduling  
- Increase equipment reliability  
- Minimize production losses
